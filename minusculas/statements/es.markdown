# Descripción

Dada una palabra $S$ con letras mayúsculas y minúsculas, has que la palabra solo tenga minúsculas.

# Entrada

Una palabra $S$.

# Salida

La palabra de entrada $S$ pero solo con minúsculas.

# Ejemplo

||input
estErnOCLeiDoMAStoiDEo
||output
esternocleidomastoideo
||end

# Límites

$|S| \leq 1000$
