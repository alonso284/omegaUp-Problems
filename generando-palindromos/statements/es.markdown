# Descripción

Dada una palabra $S$, determina cuántas letras tienes que cambiar para que se vuelva un palíndromo.

# Entrada

Una palabra $S$.

# Salida

La cantidad de letras que tienes que cambiar para que $S$ se vuelva un palíndromo.

# Ejemplo

||input
campar
||output
2
||end

# Límites

$|S| \leq 1000$
